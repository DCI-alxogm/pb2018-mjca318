Proyecto número 1.

El presente proyecto consistió en elaborar un programa(a partir de información proporcionada por la mestra), el cual nos calculará la orbita de los planetas de nuestro sistema solar al rededor del sol.

A su vez, con estos resultados elaboramos las graficas correspondientes a cada planeta.

Para lograrlo utilizamos los comandos y temas vistos a lo largo de este parcial.

Mi código consiste en lo siguiente:

*Antes de elaborar mi codigo como tal, elabore archivos de texto para cada planeta, el cual contenia una serie de datos, incluye, la información necesaria para posteriormente elaborar un archivo llamado "ResultadosPlanetax"*

1) Librerias stdio.h y math.h, las cuales me sirven de apoyo para realizar las ecuaciones del programa.

2) Doy inicio a mi programa con el int main.

3) Declaro mis variables de tiplo float, int, double y las tipo FILE correspondientes al archivo de texto.

4) Despues de esto elabore un condicional para cada planeta (if), de modo que tomaba del archivo, con sus datos correspondientes, la información necesaria para posteriormente elaborar un archivo llamado "ResultadosPlanetax"

5) Despues de haber hecho lo anterior para los 9 planetas, terminaba el programa con los {}.

Tuve complicaciones al momento de realizar las graficas, por lo cual las subi desde la computadora, guardandolas como imagen jpg.
